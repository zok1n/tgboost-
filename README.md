# TGBoost — Биржа Telegram подписок

React-приложение для биржи Telegram подписок: рекламодатели размещают задания, исполнители подписываются и получают оплату.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173 в браузере.

## Роли (демо)

В шапке сайта выберите роль:
- **Исполнитель** — просмотр заданий, выполнение подписок, баланс
- **Рекламодатель** — создание заданий, мои задания, баланс
- **Админ** — модерация, пользователи, статистика

## Технологии

- React + Vite
- Tailwind CSS
- React Router
- Zustand
- Lucide React (иконки)
